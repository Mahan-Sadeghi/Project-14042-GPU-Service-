# GPU as a Service (GaaS)

این پروژه یک سرویس شبیه‌سازی شده برای مدیریت و صف‌بندی درخواست‌های GPU است.
هدف پروژه ارائه API برای ثبت درخواست (Job)، مدیریت سهمیه کاربران و شبیه‌سازی اجرای تسک‌ها می‌باشد.

## ویژگی‌ها
- احراز هویت (JWT) و تفکیک نقش‌ها (Admin/User)
- صف‌بندی درخواست‌ها
- شبیه‌سازی اجرای GPU
- اجرا تحت Docker
